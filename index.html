Selected Projects

Semantic Code Search

This project proposed a neural model for semantic code search. When the semantics of code (e.g., identifier names) are ambiguous, previous works need to relearn the structure information from lexical tokens of code, which is extremely difficult for a model without any domain knowledge. In contrast, Our model encodes both the semantics and structures of code, by representing code with AST paths. Our model significantly outperforms both DeepCS, the first approach that applies deep learning to code search task, and CARLCS, a state-of-the-art approach that bases on deep learning. The paper is currently under submission.


Requirement-based Library Recommendation

This project identified a new perspective on library Recommendation: Recommend useful libraries given requirement descriptions of a project. It helps to avoid some key problems in existing models including requirement insufficiently matching and cold-start problem. With deep learning (DL), it is practical for models to understand the requirement of projects. We proposed a DL model for library recommendation given requirement descriptions. The experiment results show that the model can recommend libraries accurately, which proves the practicality of our perspective.
